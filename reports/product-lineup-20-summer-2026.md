## 카테고리 1: 🧹 곰팡이/위생 청소 (장마 시즌)

### ① 무향 곰팡이 제거 젤 (고점착 실리콘 틈새용)
- **한줄 컨셉**: "락스 냄새 없이, 바르고 놔두면 곰팡이가 사라진다"
- **욕구(N)**: 곰팡이=불결+건강 공포. 근데 기존 제품 냄새 때문에 청소 자체가 고역 → 숨겨진 욕구: "쉽고 안전하게 곰팡이 없애고 싶다"
- **기존 vs 우리**: 락스 기반 스프레이(5,000~8,000원, 냄새 극심) → 무향 젤(16,900원, 바르고 방치)
- **메타 후킹**: 실리콘 틈새 시꺼먼 곰팡이 → 젤 짜서 바름 → 타임랩스 30분 → 하얗게 변환 (자막: "냄새? 전혀 없어요")
- **판매가/원가/마진**: 16,900원 / 3,500원 / 약 60%
- **재구매**: 장마 시즌 1~2회 + 욕실 상시 → 연 3~4회
- **OEM 난이도**: 쉬움 (화학 제조업체 다수)
- **접시가**: **S급** — 모든 가구 필요, 건강 공포=가치 극대화
- **리스크**: 실제 무향 포뮬러 개발 가능 여부 확인 필요 (차아염소산 대안 필요)

### ② 곰팡이 방지 코팅 스프레이 (예방 특화)
- **한줄 컨셉**: "제거가 아니라, 아예 안 생기게. 한번 뿌리면 3개월 방지"
- **욕구(N)**: "매번 지워도 또 생겨" → 반복 노동에 대한 짜증. 근본 해결 욕구
- **기존 vs 우리**: 곰팡이 제거제(매번 반복) → 코팅 방지제(한번에 3개월)
- **메타 후킹**: 분할 화면 — 좌: 코팅 안 한 욕실(곰팡이 가득) vs 우: 코팅한 욕실(깨끗) + "3개월 전에 뿌렸습니다"
- **판매가/원가/마진**: 19,900원 / 4,000원 / 약 55%
- **재구매**: 3개월마다 → 연 2~4회
- **OEM 난이도**: 보통 (발수코팅+항균제 조합)
- **접시가**: **A급** — 시장에 예방 제품 거의 없음, 차별화 강함
- **리스크**: "정말 3개월 가나?" 효과 검증 필요

### ③ 에어컨 실내기 항곰팡이 필터 (붙이는 타입)
- **한줄 컨셉**: "에어컨 필터 위에 붙이기만 하면, 곰팡이균+냄새 동시 차단"
- **욕구(N)**: 에어컨 틀면 퀴퀴한 냄새 → 건강 불안 + "세척 업체 비싸"
- **기존 vs 우리**: 전문 세척 8~15만원 / 세정 스프레이 2만원(불완전) → 부착 필터 9,900원(간편)
- **메타 후킹**: 에어컨 내부 곰팡이 클로즈업(혐오) → 필터 척 붙이기 → "이걸로 끝?" → 공기질 측정기 수치 변화

- **OEM 난이도**: 쉬움 (부직포+항균 코팅)
- **접시가**: **S급** — 물길 임팩트 극강(15만원→1만원), 소모품
- **리스크**: 근본 세척 대체 못함, 보조 수단 포지셔닝 필요

### ④ 욕실 실리콘 틈새 곰팡이 방지 테이프
- **한줄 컨셉**: "곰팡이 생기는 그 틈새, 붙이면 아예 차단"
- **욕구(N)**: 실리콘 틈새 곰팡이 = 욕실 최대 스트레스. "아무리 닦아도 또 생겨"

- **메타 후킹**: 곰팡이 가득한 실리콘 위에 테이프 탁 붙임 → 깔끔 → "3개월 후에도 이 상태"
- **판매가/원가/마진**: 12,900원(3m×2롤) / 2,000원 / 약 65%
- **재구매**: 3~6개월마다
- **OEM 난이도**: 쉬움 (방수테이프+항균 소재)
- **접시가**: **A급** — 심리적 쾌감(깔끔해 보임), 설치 1분

- **OEM 난이도**: 쉬움 (탈취 파우더)

- **접시가**: **A급** — 극소량 노력, 저가 소모품
- **리스크**: 근본 세척 아님, "향으로 덮는 것 아니냐" 인식

### ⑮ 실내 습기+냄새 동시 제거기 (제습+탈취 올인원)
- **한줄 컨셉**: "장마철 눅눅함+퀴퀴한 냄새, 이거 하나로 동시 해결"
- **욕구(N)**: 장마=습기+냄새 복합 스트레스. 제습기(비쌈)+탈취제(따로 사야) → 원스톱 해결
- **기존 vs 우리**: 물먹는 하마(습기만) + 방향제(냄새만) → 제습+탈취+항균 올인원 파우치
- **메타 후킹**: 습도계 80% → 파우치 놓기 → 24시간 후 습도 55% + "냄새도 사라졌어요"
- **판매가/원가/마진**: 14,900원(4개 세트) / 3,000원 / 약 55%
- **재구매**: 2~3개월마다
- **OEM 난이도**: 쉬움 (실리카겔+활성탄+항균제)
- **접시가**: **A급** — 복합 해결, 가성비
- **리스크**: 물먹는 하마 브랜드 인지도와 경쟁

### ⑯ 신발장 전용 제습·탈취 캡슐 (넣어두면 3개월)

## 카테고리 5: 👕 빨래/세탁 (장마 실내건조)

### ⑰ 빨래 쉰내 킬러 캡슐 (세탁기 투입형, 모락셀라균 99.9%)
- **한줄 컨셉**: "세탁기에 1캡만 넣으세요. 쉰내 원인균 99.9% 제거"
- **욕구(N)**: "장마철 빨래에서 쉰내" = 수치심(내 옷에서 냄새). 섬유유연제로 안 됨 → 좌절
- **기존 vs 우리**: 섬유유연제(향으로 덮기만) / 과탄산소다(담그기 귀찮) / 건조기(80만원) → 캡슐 1개(원인균 제거)
- **메타 후킹**: 실내 건조한 수건 냄새 맡고 인상(공감) → 캡슐 1개 투입 → 건조 후 냄새 테스트 → 미소
- **판매가/원가/마진**: 16,900원(30캡/1달분) / 3,000원 / 약 60%
- **재구매**: 매달
- **OEM 난이도**: 보통 (세제 OEM + 항균 원료)
- **접시가**: **S급** — 물길 임팩트(건조기 80만→1.7만), 매달 소모
- **리스크**: "기존 항균 세제와 뭐가 다르냐" 차별화 전달

### ⑱ 세탁조 클리너 + 곰팡이 방지제 (올인원)
- **한줄 컨셉**: "세탁기 자체가 곰팡이 덩어리인 거 아셨나요? 한 번에 세척+방지"
- **욕구(N)**: "세탁기가 더러우면 빨래도 더럽다" → 공포 마케팅 가능
- **기존 vs 우리**: 세탁조 클리너(세척만) → 세척+코팅 방지(다음 장마까지 보호)
- **메타 후킹**: 세탁조 안쪽 곰팡이/찌꺼기 클로즈업(혐오) → 투입 → 검은 물 나옴 → "이게 다 내 옷에..."
- **판매가/원가/마진**: 12,900원(3회분) / 2,000원 / 약 65%
- **재구매**: 월 1회 → 연 12회
- **OEM 난이도**: 쉬움
- **접시가**: **A급** — 공포+혐오 = 즉각 전환, 소모품
- **리스크**: 옥시/세탁조 클리너 기존 브랜드와 경쟁

### ⑲ 빨래 건조 촉진 스프레이 (실내건조 시간 50% 단축)
- **한줄 컨셉**: "실내건조해도 쉰내 없는 비결 — 건조 시간을 반으로"

- **판매가/원가/마진**: 14,900원 / 3,000원 / 약 55%
- **재구매**: 장마 시즌 2~3개
- **OEM 난이도**: 어려움 (건조 촉진 포뮬러 개발)
- **접시가**: **B급** — 컨셉 참신하지만 효과 검증 난이도 높음
- **리스크**: "진짜 빨리 마르나" 효과 의심, 포뮬러 개발
