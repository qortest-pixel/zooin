### 1위. 🧹 곰팡이제거제 — "장마철 국민 고통" (접시가: S)

#### 욕구(N) 분석
- **숨겨진 욕구**: "우리집이 더럽다"는 걸 남에게 말 못함 → **욕구가 숨기를 좋아한다** 완벽 해당
- **욕구 강도**: ⭐⭐⭐⭐⭐ (욕구깡패급). 건강 공포(곰팡이=폐질환) + 청결 강박
- **욕구 흐름**: 장마 시작 → 곰팡이 발견 → 패닉 → 즉시 구매 (사고력 낮아진 상태)
- **타겟**: 자취생, 신혼부부, 맞벌이 주부 (20~40대)

#### 소비자 불만/페인포인트 (실제 후기 기반)
1. **"냄새가 너무 독해서 머리 아파요"** — 락스(차아염소산나트륨) 기반 = 냄새 지옥
2. **"뿌리고 닦아도 금방 다시 생겨요"** — 표면만 제거, 근본 원인(습기) 미해결
3. **"벽에 뿌렸더니 젤이 흘러내려요"** — 점착성 부족, 수직면에서 효과 반감
4. **"깊게 박힌 곰팡이는 안 없어져요"** — 오래된 곰팡이에 무력
5. **"아이/반려동물 있으면 쓰기 무서워요"** — 화학 성분 안전성 우려

#### 시장의 빈틈 (경쟁자가 안 하는 것)
- ✅ **"냄새 없는 곰팡이제거제"** → 시장 공백! 대부분 락스 냄새 강함
- ✅ **"뿌리고 놔두면 끝"** → 닦아야 하는 번거로움 제거
- ✅ **"곰팡이 예방"까지 되는 올인원** → 제거+방지 콤보 거의 없음
- ✅ 아이/반려동물 안전 인증 제품 부족

#### CV(전환) 포인트 — 우리가 찌를 곳
> **"무향·저자극 곰팡이제거 + 방지 젤"**
> - 후킹 영상: 시꺼먼 곰팡이 → 젤 바르고 30분 → 하얗게 (Before/After 타임랩스)
> - 차별화: "락스 냄새 0% / 아이 있는 집도 OK"
> - 가격대: 14,900~19,900원 (세트)
> - 메타 3초 후킹: "이 곰팡이가 30분 만에 사라집니다" + 충격적 비포 영상

## 🎯 M군님이라면? — 최종 전략 추천

### M군님 방식 시장 진입 플레이북

#### Step 1: 욕구(N)에서 출발 — "R에 갇히지 마라"
> 제품을 먼저 정하지 말고, **"여름에 사람들이 가장 못 참는 것"**에서 시작

**여름 욕구 히트맵:**
```
못 참는 정도 (욕구깡패 순)
━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 체취/땀     ★★★★★ (사회적 죽음 공포)
2. 곰팡이/위생  ★★★★★ (건강 공포)
3. 빨래 쉰내   ★★★★☆ (수치심)
4. 더위       ★★★★☆ (즉각 불편)
5. 벌레       ★★★★☆ (공포+혐오)
```

#### Step 2: 물길 임팩트 찾기 — "비싼 것의 저렴한 대안"

## 💰 최종 추천: 동시 진입 2+1 전략

### 🎯 주력 2개 (동시 런칭)
1. **곰팡이제거제 (무향·저자극 젤)** — 장마 직전 5~6월 런칭
2. **땀패치+데오도란트 세트** — 5월 초여름부터

### 🧪 테스트 1개
3. **쿨링스프레이 (30분 지속·무알코올)** — 6월 폭염 시즌 테스트

### 왜 이 조합인가? (M군님 기준)
- **곰팡이**: "욕구가 숨기를 좋아한다" + Before/After 영상 = 메타 최적
- **땀패치**: "욕구깡패" + 매일 소모 = 재구매 LTV 최고
- **쿨링**: "독점 브랜드 없음" + 시각적 시연 극강 = 신규 진입 블루오션

### 타임라인
```
4월: OEM 컨택 + 샘플 (3개 동시)
5월 초: 땀패치 선행 런칭 → 픽셀 학습
5월 말: 곰팡이제거제 런칭 (장마 직전)
6월: 쿨링스프레이 테스트 런칭
6~8월: 3개 동시 운영, 주간 ROAS 리뷰
```
